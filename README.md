## 📌 Project Overview

이번 프로젝트에서는 하나의 가상 브랜드를 직접 기획하고, AI 생성 도구를 활용하여 **10초 이내의 브랜드 광고 영상**을 제작했습니다.

광고 제작 과정에서 이미지, 비디오, 오디오 생성 AI를 조합하여 하나의 완성된 브랜드 경험을 구현했으며, 각 단계에서 사용한 프롬프트와 도구 선택 이유를 함께 기록했습니다.

프로젝트의 핵심은 **좋은 결과물을 만드는 것**보다 **기획 의도를 AI가 이해할 수 있는 프롬프트로 변환하고, 원하는 결과가 나올 때까지 반복적으로 개선하는 과정**에 있습니다.

---

## 🎯 Project Objectives

본 프로젝트의 목표는 다음과 같습니다.

- 브랜드 아이덴티티를 정의하고 광고를 기획한다.
- 생성형 AI를 활용하여 이미지, 영상, 오디오를 제작한다.

## 🎯 Brand Vision

EcoDrop은 일회용품 사용을 줄이고 지속 가능한 소비 문화를 확산하기 위해 만들어진 친환경 라이프스타일 브랜드입니다.

## ⚙️ Chapter 4. AI Production Pipeline

> **"좋은 결과물은 좋은 프롬프트보다, 좋은 제작 파이프라인에서 시작됩니다."**

광고 제작은 기획 단계에서 브랜드의 핵심 메시지를 정의한 후, 이를 프롬프트로 구체화하고, 생성된 결과물을 통합하여 최종 영상으로 완성하는 순서로 진행되었습니다.

## 📝 Step 1. Brand Planning

광고 제작을 시작하기 전에 브랜드의 방향성을 먼저 정의했습니다.

먼저 브랜드의 철학, 타깃 고객, 톤앤매너를 설정하고, 이를 기반으로 광고의 핵심 메시지를 도출했습니다.

브랜드의 정체성이 명확해야 이후 생성되는 이미지와 영상의 스타일도 일관성을 유지할 수 있기 때문입니다.

## 📖 Step 2. Storyboard

스토리보드는 AI 생성 이전에 광고의 전체 흐름을 설계하는 단계입니다.

각 씬마다 다음 항목을 미리 정의했습니다.

- Scene Number
- Duration
- Goal Message
- Camera Composition
- Narration
- AI Tool
- Prompt
- Expected Output

이를 통해 생성 과정에서 불필요한 재생성을 줄이고, 제작 효율을 높일 수 있었습니다.

## 💬 Step 3. Prompt Engineering

스토리보드를 기반으로 각 씬에 맞는 프롬프트를 작성했습니다.

광고의 분위기와 브랜드 이미지를 유지하기 위해 모든 프롬프트에서 다음 요소를 일관되게 유지했습니다.

### Prompt Strategy

- Cinematic Lighting
- Premium Product Photography
- Minimal Composition
- Natural Color Palette
- Commercial Advertising Style
- High Detail
- Ultra Realistic

## 💬 Chapter 7. Prompt Engineering

> **"좋은 결과물은 좋은 프롬프트에서 시작됩니다."**

이번 프로젝트에서는 단순히 AI에게 이미지를 생성하도록 요청하는 것이 아니라, **브랜드의 아이덴티티와 광고의 목적을 반영한 프롬프트를 설계**하는 데 중점을 두었습니다.

광고의 모든 Scene에서 일관된 스타일을 유지하기 위해 공통 키워드를 정의하고, 각 Scene의 목적에 맞는 세부 요소를 추가하는 방식으로 프롬프트를 구성했습니다.

---

# 🎯 Prompt Design Strategy

프롬프트를 설계할 때 다음과 같은 원칙을 적용했습니다.

- 브랜드의 톤앤매너를 유지한다.
- 광고 스타일의 시네마틱 연출을 강조한다.
- 제품이 자연스럽게 중심이 되도록 구성한다.
- 불필요한 요소를 최소화하여 AI의 해석 범위를 줄인다.
- 모든 Scene에서 동일한 스타일 키워드를 사용하여 일관성을 유지한다.

---

# 🧩 Prompt Structure

이번 프로젝트에서 사용한 프롬프트는 다음과 같은 구조를 기반으로 작성했습니다.

```text
Subject
    ↓
Environment
    ↓
Lighting
    ↓
Camera Composition
    ↓
Style
    ↓
Quality
```

예를 들어 하나의 프롬프트는 다음과 같은 순서로 작성했습니다.

```text
Premium reusable stainless tumbler,
minimal product photography,
green background,
soft natural lighting,
commercial advertising style,
cinematic composition,
ultra realistic,
8K
```

# 📌 Prompt Consistency

브랜드 광고에서는 모든 Scene이 하나의 영상처럼 느껴져야 합니다.

이를 위해 다음 요소를 고정하여 사용했습니다.

- Commercial Advertising Style
- Minimal Composition
- Soft Natural Lighting
- Cinematic Camera
- Premium Product Photography
- Ultra Realistic
- High Detail

이러한 공통 키워드를 유지함으로써 Scene마다 생성된 결과물이 서로 다른 이미지가 아니라 하나의 광고처럼 연결될 수 있도록 했습니다.

# 💡 Key Insight

프롬프트는 단순히 AI에게 명령을 내리는 문장이 아니라, **브랜드의 기획 의도를 AI가 이해할 수 있는 언어로 번역하는 과정**이라고 생각했습니다.

따라서 모든 프롬프트는 장면의 목적과 브랜드의 핵심 메시지를 반영하도록 설계했으며, 결과물을 반복적으로 확인하고 수정하는 과정을 통해 완성도를 높였습니다.

## 🔄 Chapter 8. Prompt Refinement

> **"첫 번째 프롬프트가 최고의 결과를 만드는 경우는 거의 없습니다."**

생성형 AI는 동일한 의도를 입력하더라도 프롬프트의 표현 방식에 따라 전혀 다른 결과를 생성할 수 있습니다.

이번 프로젝트에서는 원하는 광고 스타일을 얻기 위해 생성 결과를 반복적으로 분석하고, 문제점을 개선하면서 프롬프트를 수정했습니다.

이 과정은 단순한 재생성이 아니라 **결과를 평가하고 개선하는 프롬프트 엔지니어링 과정**으로 진행되었습니다.

# 🎬 Case Study — Scene 02

## 목표

EcoDrop 텀블러를 브랜드 광고처럼 자연스럽게 강조하는 제품 컷을 생성한다.

---

## 📝 Initial Prompt

```text
A young woman holding a reusable tumbler,
green background,
natural lighting
```

---

## ⚠️ Problems

생성 결과를 확인한 결과 다음과 같은 문제가 발생했습니다.

- 제품보다 인물이 화면의 중심이 됨
- 텀블러의 디테일이 부족함
- 광고보다 일반 스냅사진처럼 표현됨
- 배경 스타일이 일정하지 않음
- 브랜드의 프리미엄 이미지가 잘 드러나지 않음

---

## 🔍 Analysis

초기 프롬프트는 **주제(Subject)**만 설명하고 있었으며,

광고 스타일을 결정하는

- Composition
- Camera
- Lighting
- Commercial Style

등의 요소가 포함되지 않았습니다.

그 결과 AI는 일반적인 인물 사진으로 해석하여 광고 의도와 다른 이미지를 생성했습니다.

---

## ✨ Refined Prompt

```text
Premium reusable stainless tumbler,
product-centered composition,
minimal green background,
commercial advertising photography,
soft cinematic lighting,
high-end product showcase,
ultra realistic,
8K,
sharp focus
```

---

## 📈 Improvements

프롬프트 수정 후 다음과 같은 변화가 나타났습니다.

| Before | After |
|----------|---------|
| 인물 중심 | 제품 중심 |
| 일반 사진 | 광고 스타일 |
| 배경 불안정 | 미니멀한 배경 |
| 낮은 제품 강조 | 브랜드 중심 구성 |
| 분위기 불일치 | 브랜드 톤앤매너 유지 |

# 🚀 Iterative Improvement

프롬프트는 한 번에 완성되지 않았으며, 생성 결과를 확인한 뒤 반복적으로 수정했습니다.

```text
Prompt v1
    │
    ▼
Generate
    │
    ▼
Evaluate
    │
    ▼
Revise
    │
    ▼
Prompt v2
    │
    ▼
Generate Again
    │
    ▼
Final Result
```

이러한 반복 과정을 통해 브랜드가 의도한 메시지와 광고의 시각적 완성도를 점차 높일 수 있었습니다.

- 스타일

- 원하는 결과를 얻기 위해서는 반복적인 수정과 검증 과정이 필수적이라는 것을 확인했습니다.

---

### 🤖 Multimodal AI Workflow

이번 프로젝트에서는 하나의 AI만 사용하는 것이 아니라,

- 이미지 생성 AI
- 영상 생성 AI

이를 통해 각 AI의 역할과 장점을 이해하고, 서로 다른 생성 결과를 하나의 콘텐츠로 통합하는 경험을 할 수 있었습니다.

# 🚧 Challenges

프로젝트를 진행하면서 다음과 같은 어려움이 있었습니다.

### ⚠️ Style Consistency

Scene마다 생성 결과의 분위기가 달라지는 문제가 발생했습니다.

### ✔️ Solution

---

### ⚠️ Credit Consumption

영상 생성 과정에서는 많은 크레딧이 소모되었습니다.

### ✔️ Solution

스토리보드를 먼저 확정하고 이미지를 충분히 검토한 후 영상으로 변환하여 재생성을 최소화했습니다.

---

### ⚠️ Prompt Accuracy

초기 프롬프트에서는 원하는 광고 분위기가 잘 표현되지 않았습니다.

### ✔️ Solution

카메라 구도, 조명, 스타일, 제품 중심 구성과 같은 요소를 구체적으로 명시하여 결과의 품질을 향상시켰습니다.

## 📎 Appendix. Production Standards & Technical Decisions

> **"이 프로젝트는 단순 생성이 아니라, 재현 가능한 제작 시스템을 목표로 설계되었습니다."**

# 📁 1. Asset Naming Convention

모든 생성 결과물은 다음과 같은 규칙으로 관리하였습니다.

```text
scene{번호}_{type}_{version}.ext
```

### Example

- scene01_image_v1.png
- scene01_video_v1.mp4
- scene02_audio_v1.wav

### Purpose

- 파일 재사용성 확보
- Scene 단위 관리 용이
- 버전 비교 및 프롬프트 개선 추적 가능

---

# 🎨 2. Text-to-Image vs Image-to-Video Strategy

본 프로젝트에서는 다음 기준으로 생성 방식을 분리하여 사용했습니다.

| 방식 | 사용 목적 | 선택 이유 |
|------|----------|-----------|
| Text-to-Image | Key Visual 제작 | 광고 콘셉트를 가장 정확하게 반영 가능 |
| Image-to-Video | 모션 생성 | 일관된 스타일 유지 및 비용 절감 |

### Decision Principle

- 초기 콘셉트 정의 → Text-to-Image 사용
- 최종 광고 컷 → Image-to-Video 사용
- 불필요한 변형 최소화 전략 적용

---

# ⚖️ 3. Tool Selection Criteria

도구 선택은 다음 4가지 기준으로 평가하여 결정했습니다.

| 기준 | 설명 | 우선순위 |
|------|------|----------|
| 🎨 Quality | 광고 퀄리티 수준 | 1순위 |
| 🎯 Control | 프롬프트 제어 가능성 | 2순위 |
| ⚡ Speed | 생성 속도 | 3순위 |
| 💰 Cost | 크레딧/비용 효율 | 4순위 |

### Conclusion

- 광고 프로젝트 특성상 **Quality + Control**이 가장 중요
- Speed와 Cost는 보조 요소로 고려

---

# 🎭 4. Style & Character Consistency Strategy

광고 전체에서 시각적 일관성을 유지하기 위해 다음 방법을 사용했습니다.

### 1) Prompt Locking

모든 Scene에서 동일한 핵심 키워드를 고정 사용:

- cinematic lighting  
- commercial advertising style  
- minimal composition  
- ultra realistic  
- soft natural tone  

---

### 2) Style Anchoring

각 Scene에서 스타일이 흔들리지 않도록 동일한 표현 구조 유지:

```text
Subject → Environment → Lighting → Camera → Style → Quality
```

---

### 3) Reference Consistency (Pseudo Control)

- 동일한 제품 설명 유지
- 동일한 색감 키워드 반복 사용
- Scene 간 분위기 연결 유지
